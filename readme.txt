Show gravatar on your widget and replace your Recent COmments with this one.

== Description ==

Allows you display your recent commentator's gravatar. It just like myBlogLog's widget. The difference is just the avatar.

== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure Recent Gravatar on your Widget Settings
